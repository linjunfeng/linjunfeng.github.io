js 正则总结

String 对象

search 起始位置，不存在返回-1

"1234abc".search(/abc/);//4
"1234abc".search(/abcd/);//-1

replace 替换

"hello, world".replace(/world/i, "php");//hello, php

match 匹配

"1,2,3".match(/\d+/);//返回也是数组,a[0]就是匹配的结果,依次子表达式的结果
"1,2,3".match(/\d+/g);//["1", "2", "3"]

RegExp 对象

test

/world/.test("hello, world");//true

exec

//exec 和 test 一样。 只是有返回结果， 结果跟 String.match 一样
/world/.exec("hello, world");
